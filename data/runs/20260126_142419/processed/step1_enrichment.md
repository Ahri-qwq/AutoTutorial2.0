# ABACUS Metadata: Smearing (高斯展宽) 参数影响分析

## 1. 物理本质 (Physics Concepts)
- **核心物理概念**: 
    - **电子占据函数平滑化 (Smoothing of Occupation Function)**: 在费米能级（Fermi Level）附近引入一种数学上的展宽机制，将原本在 0K 下的阶梯状电子占据函数（Step Function）替换为平滑函数（如高斯函数、Fermi-Dirac 函数等）。
    - **电子熵 (Electronic Entropy)**: Smearing 的引入会在总能量中产生额外的电子熵项（$-TS$），使得计算出的自由能与真实的基态能量存在偏差。
- **解决的科学问题**:
    - **收敛性问题**: 主要解决金属体系或费米面附近能级密集、复杂的体系在 SCF 迭代中电荷密度震荡、难以收敛的问题。
    - **分数占据**: 允许电子以分数形式占据轨道，从而更准确地描述费米面附近的电子态。

## 2. 关键输入参数 (Key Parameters)

### INPUT 文件参数

| 参数名 | 推荐值 / 常用值 | 物理意义与说明 |
| :--- | :--- | :--- |
| **smearing_method** | `gauss` (默认/通用)<br>`mp` (金属推荐)<br>`fixed` (绝缘体) | **展宽方法类型**。<br>- `fixed`: 固定占据，仅适用于非导体（绝缘体）。<br>- `gauss` / `gaussian`: 高斯展宽，默认方法，适用于大多数体系（包括半导体）。<br>- `mp` / `mp2`: Methfessel-Paxton 方法（1阶或2阶），推荐用于**金属**体系。<br>- `fd`: Fermi-Dirac 方法，模拟有限温度，半导体/非导体可用。<br>- `mv` / `cold`: Marzari-Vanderbilt 方法。 |
| **smearing_sigma** | `0.01` (常用)<br>`0.015` (默认) | **展宽能量范围**。<br>- 单位：**Ry (Rydberg)**。<br>- 控制占据函数的“弥散”程度。数值越大，收敛越容易，但结果偏离基态越远。<br>- 注意：对于 MD 模拟，有时使用更小的值（如 `0.001` Ry）。 |
| **smearing_sigma_temp** | *需查阅文档确认推荐值* | **基于温度的展宽控制**。<br>- 资料提及该参数可替代 `smearing_sigma`，但不可同时使用。<br>- 物理意义类似，但可能直接以温度为单位控制展宽。 |

### 【重要】知识缺口处理
- **`smearing_sigma_temp` 的单位**: 资料3 提到了该参数，但未明确其输入单位是 Kelvin 还是其他能量单位。**撰写时请务必提醒读者查阅官方文档确认单位**，切勿假设。
- **DOS 计算的关联**: 资料6 提到了 `dos_sigma` (单位 eV)，这是后处理 DOS 的展宽，需与 SCF 计算用的 `smearing_sigma` (单位 Ry) 严格区分。

## 3. 体系与接口配置 (System & Interfaces)

- **结构 (STRU)**: 
    - 无特殊结构要求。
    - 对于**金属体系**或**小带隙半导体**，Smearing 设置尤为关键。
- **外部接口**: 
    - 无直接外部工具依赖。
- **接口注意事项**: 
    - 在进行能带结构或 DOS 分析时，SCF 步骤中使用的 Smearing 参数会影响费米能级的位置，进而影响后续性质分析的基准。

## 4. 教程编写特殊指令 (Special Instructions for Writer)

- **Critical (核心强调点)**:
    1.  **单位陷阱**: 必须反复强调 `smearing_sigma` 的单位是 **Ry** (Rydberg)，而不是 eV。许多从 VASP 转来的用户习惯用 eV，直接填入 `0.05` (Ry) 会导致展宽过大（约 0.68 eV），严重影响结果。
    2.  **能量对比原则**: 资料5 明确指出，如果对比不同体系的总能量（如计算形成能、吸附能），**必须保持 Smearing 参数一致**。严禁为了某个难收敛的结构单独增大 Smearing，这会导致能量基准不一致。
    3.  **收敛与精度的权衡**: 解释 "Sigma 越大越好收敛，但能量越不准" 的 Trade-off。建议教程中包含一个策略：先用较大的 Sigma (如 0.02 Ry) 跑粗略收敛，再逐步减小 Sigma (至 0.01 Ry 或更小) 获得精确能量。
    4.  **半导体 vs 金属**: 明确区分：
        - 宽带隙绝缘体/半导体：推荐 `fixed` 或 `gauss` (小 sigma)。
        - 金属：推荐 `mp` 或 `mp2`。

- **风险提示**: 
    - 资料6 中出现了 `smearing_sigma 0.6` 配合 `fd` 的用法（可能是随机波函数 DFT 的特殊情况或高温模拟），这与常规 DFT 的 `0.01` Ry 差异巨大。撰写常规 DFT 教程时，请以 `0.01` - `0.02` Ry 为主，避免读者混淆。

## 5. 常见报错与注意事项 (Pitfalls)

- **总能量不准确**: 
    - 现象：`smearing_sigma` 设置过大（例如 > 0.05 Ry）。
    - 后果：虽然 SCF 很快收敛，但计算出的总能量包含了过大的电子熵贡献，不再代表零温基态能量。
- **参数互斥报错**:
    - 同时设置了 `smearing_sigma` 和 `smearing_sigma_temp` 会导致程序报错或行为未定义。
- **不收敛误区**:
    - 即使增大了 Smearing 仍不收敛时，不要无限增大该值。应检查 `mixing_beta` (电荷密度混合参数) 或结构本身的合理性。
- **受力收敛滞后**:
    - 资料5 提示：总能量收敛时，受力和应力可能尚未完全收敛。对于弛豫计算，需注意 Smearing 对力的影响。
